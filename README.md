A mobile application designed to streamline volunteer coordination during disasters in Kenya by providing a centralized, real-time platform for emergency requests, volunteer matching, and incident reporting.

## Problem Statement

Disaster response in Kenya often relies on fragmented communication channels such as WhatsApp groups and phone calls, leading to delayed responses, duplicated efforts, and poor visibility of available volunteer skills. This system addresses these challenges through a unified digital solution.

## Project Objectives

Centralize disaster response volunteer coordination

Match volunteers to emergencies based on skills, availability, and location

Enable real-time emergency alerts and incident reporting

Improve response efficiency and situational awareness

## Key Features

User Authentication (Firebase Authentication)

Volunteer Profiles (skills, availability, service area)

Emergency Request Creation (location, urgency, required skills)

Volunteer Matching System

Disaster Alerts Dashboard

Real-Time Incident Updates with image uploads

Availability Toggle for volunteers
